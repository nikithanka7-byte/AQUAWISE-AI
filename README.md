### Aqua Wise AI 💧

Smart Water Conservation & SDG 6 Awareness Platform

### Project Overview

Aqua Wise AI is an AI-powered web application designed to promote water conservation, educate users about Sustainable Development Goal 6 (SDG 6), and provide intelligent recommendations for efficient water usage. The platform integrates Groq's Large Language Models (LLMs) with Streamlit to deliver real-time guidance, educational content, quizzes, and water-saving suggestions.

The project aims to create awareness about responsible water management while helping users make informed decisions regarding water conservation and sanitation practices.

### Installation

Step 1: Clone the Repository

git clone https:https://github.com/nikithanka7-byte/AQUAWISE-AI

Step 2: Install Dependencies

pip install -r requirements.txt

Or manually install:

pip install streamlit
pip install groq


---

### Groq API Setup

Create a Groq Account

Generate API Key

1. Sign in to Groq Console.


2. Navigate to API Keys.


3. Create a new API key.


4. Copy and securely store the key.




---

### Configuration

Add your API key in app.py:

from groq import Groq

client = Groq(
    api_key="YOUR_GROQ_API_KEY"
)


---

### Running the Application

streamlit run app.py

The application will launch in your default browser.
